# opencode-assets

自用的 opencode 资源库，包含自定义 skills 和 agents。

## 目录结构

- \skills/\ - 自定义技能
- \gents/\ - 自定义代理

## Skills

| Skill | 说明 |
|-------|------|
| skill-creator | 创建新 Skill（支持搜索现有 Skill） |
| skill-update | 迭代优化现有 Skill |
| agent-creator | 创建 OpenCode Agent（支持搜索现有 Agent） |
| agent-update | 迭代优化现有 Agent |
| model-guide | 模型选择指南（34 个 Zen 模型 + 场景推荐） |
| glue-coding | 胶水编程模式 |
| architecture-spec | 架构可视化交互规范 |
| canvas-dev | Canvas 白板驱动开发 |

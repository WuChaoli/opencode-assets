## 快速创建流程

1. 沟通确认需求
2. **搜索现有 Agent**（Claude Code Market / awesome-opencode / GitHub）
3. **评估搜索结果**（匹配度 / 质量 / 可改编性）
4. 在线调研最佳实践
5. 设计 Agent 方案
6. 与用户确认方案
7. 确认创建位置
8. 执行创建/改编
9. 质量验证
10. 交付与说明

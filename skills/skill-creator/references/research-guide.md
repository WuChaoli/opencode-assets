# 在线调研指南

## 调研目标

在创建 Skill 之前，通过在线调研获取：
1. 官方文档和社区最佳实践
2. 类似 Skill 的实现方案
3. 常见陷阱和反模式
4. 可复用的代码片段和模板

## 调研步骤

### 1. 搜索官方文档
- 访问官方文档站点
- 提取核心 API/命令/工作流
- 记录版本兼容性要求

### 2. 搜索社区经验
- GitHub Issues 中的常见问题
- Stack Overflow 高票回答
- 博客文章和教程
- 社区论坛讨论

### 3. 搜索类似实现
- 搜索已有的 Skill 实现
- 分析其目录结构和内容组织
- 提取可复用的模式

### 4. 整理调研结果
- 按主题分类整理
- 标注来源和可信度
- 提炼可复用的核心模式

## 调研关键词建议

| 类型 | 关键词示例 |
|------|-----------|
| 官方文档 | `[库名] documentation`, `[库名] API reference` |
| 最佳实践 | `[库名] best practices`, `[库名] tips` |
| 常见问题 | `[库名] common issues`, `[库名] troubleshooting` |
| 类似实现 | `[库名] skill`, `[库名] claude skill`, `[库名] agent skill` |

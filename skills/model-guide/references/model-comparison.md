# 场景化模型推荐

## 1. 代码开发类

### 功能需求
- **编码能力**: 生成高质量、可维护、符合规范的代码
- **上下文理解**: 理解项目结构、依赖关系、业务逻辑
- **工具调用**: 稳定调用 Read/Write/Edit/Bash 等工具
- **架构思维**: 理解设计模式、分层架构、接口设计
- **代码规范**: 遵循语言最佳实践和团队约定

### 包含场景
新功能开发、代码补全、代码重构、代码优化、模板生成
